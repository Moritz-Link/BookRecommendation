Book Recommendation

This is a project based on the idea to use Graph Neural Networks for Recommendation Systems. In this work a GraphSAGE is used to recommend up to five books to a new user, depending on his selection.
Right now the application is very basic and missing lot of functions like a search or a load more button.
I will incorporate these functions over time.


Application
- Choose up to 5 books
- Click predict

Data
I used the Book-Crossing dataset mined by Cai-Nicolas Ziegler
(http://www2.informatik.uni-freiburg.de/~cziegler/BX/)
